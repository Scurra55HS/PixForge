# PixForge

PixForge é um editor de pixel art simples e intuitivo, feito em **HTML, CSS e JavaScript puro**, onde você pode criar desenhos pixel por pixel escolhendo diferentes tamanhos de grid.

A ideia do projeto é unir criatividade, lógica e manipulação de DOM em um app leve, rápido e divertido.

---

## Funcionalidades

- Escolha de grid (8x8, 16x16, 32x32)
- Pintura de pixels com clique ou toque
- Borracha para apagar pixels
- Seletor de cores
- Suporte a desktop e mobile
- Interface simples e responsiva

---

## Tecnologias usadas

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

Sem frameworks. Sem bibliotecas externas.

---

## Objetivo do projeto

- Treinar JavaScript puro
- Trabalhar eventos de mouse e touch
- Manipular o DOM de forma dinâmica
- Criar um projeto criativo e utilizável para portfólio
